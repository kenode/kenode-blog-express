# Kenode Blog Express

------

一个基于 __Node.js__ 的博客实践

## 运用的技术

### 1. 后端

> * express 4.x  -- 基础框架
> * mongodb -- NoSql数据库
> * redis -- 缓存数据
> * swig -- 模版引擎
> * log4js -- 日志处理
> * bluebird -- 异步流程控制库，基于Promises/A+

### 2. 前端

> * jquery -- 运用最广泛的Javascript库
> * bootstrap 3.x -- 可视化布局视图框架
> * react -- 来自Facebook的组件化开发框架
> * lodash -- javscript工具库

### 3. 开发编译

> * gulp -- 自动化构建工具
> * webpack -- 模块加载打包工具
> * babel -- ES6转换ES5工具
> * sass -- css预编译器
